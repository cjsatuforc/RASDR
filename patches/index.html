Notes

Additional information regarding the above patches is provided below:

  • RASDRproc-v0.2.3.5
    1. Integrates with Jim Sky's Radio Sky Spectrograph
    2. Adds DC offset correction and control on the front panel.
    3. Fixes some quirky behavior with FFT spectrum markers and displays their relative position correctly.

    1A1B,1C2
  • RASDRproc-v0.2.3.8
    1. UI and Control Panel Enhancements
    2. Adds control of RX input selector on the front panel.
    3. Provides immediate control over as many parameters as possible without going thru a stop/start cycle. Those that are not allowed to be changed without a stop/start are greyed out. If you can change it, the effect occurs immediately.
    4. Background subtraction controls are available in the Performance Parameters->Tuning Parameters.
    5. Update graphs actually works correctly now; processing and file/RSS output continue even though UI display does not update. This can lead to significant more data processing on low-spec systems.
    6. Interprets RASDR serial number and provides tooltip information for many selections.
    7. Enhancements to auto-restart which helps low-spec systems keep taking data and not glitch.

    34A4B
