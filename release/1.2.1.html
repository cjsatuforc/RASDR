Release Notes - 1.2.1

The 1.2.1 release of RASDR is the first unified release of all the software developed for the RASDR project to date. All the source code is maintained for developers on the github repository.

Through the generous support of Lime Microsystems, Ltd., members of the Society of Radio Amateur Astonomers are able to use several prototype hardware modules to develop a software package to enable pre-production evaluation of the specifications and capabilities of the RASDR hardware.

Download

Please obtain the distribution as a single file download: rasdr-distribution-1.2.1.zip

You can browse individual files if you wish.

Features

The following modules are available in this distribution:

  • Signed Windows 7/Vista/XP device driver
  • Firmware to support the signed driver
  • A tool to program the firmware on the board (courtesy of Cypress Semiconductor)
  • RASDRviewer, the configuration, visualization and data collection tool
  • plotcsv, a utility to post-process data files obtained by RASDRviewer
