Release Notes - 1.2.4

The 1.2.4 release of RASDR is the fourth release of all the software developed for the RASDR project to date. All the source code is maintained for developers on the github repository.

Through the generous support of Lime Microsystems, Ltd., members of the Society of Radio Amateur Astonomers are able to use several prototype hardware modules to develop a software package to enable pre-production evaluation of the specifications and capabilities of the RASDR hardware.

Features

The following modules are available in this distribution:

  • WHQL Windows 10/Windows 8/Windows 7/Vista/XP device driver
  • Firmware to support the driver (with fix to USB3 operation)
  • A tool to program the firmware on the board (courtesy of Cypress Semiconductor)
  • RASDRproc, the configuration, visualization and data collection tool
  • plotcsv, a utility to post-process data files obtained by RASDRproc and RASDRviewer
  • RASDRstreamer, a tool to capture RAW data from a RASDR device
  • raw2iq, a utility to post-process RAW data files obtained by RASDRstreamer
